# Cache Integration Tests

## Setup (run once)

```bash
# 1. Install Dart dependencies
dart pub get

# 2. Generate the Dart client from the schema
make generate

# 3. Build the Spring Boot server jar
make build-server
```

## Running the tests

```bash
# Start the server in a separate terminal
make run-server

# Then in another terminal
make test
```

## Rules

### Generated Code — NEVER TOUCH
Never read, edit, create, or delete any file under `lib/generated/`.
These files are auto-generated by GraphLink and any manual change will be lost on the next generation.

### Schema consistency
`schema/schema.gql` (client, with graphlink directives) and
`spring-server/src/main/resources/graphql/schema.graphqls` (server, plain GraphQL)
must declare the same types, queries, and mutations.
If you change one, update the other.
